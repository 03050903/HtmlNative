HtmlNative
========

使用Html来渲染Android界面的动态化界面方案。

不同于Webview, HtmlNative直接将Html和Css转换成原生控件，这样做的好处是：

- 让界面能达到更好的体验效果
- 用户无感知
- 简易部署，使用最容易的html + css来控制界面
